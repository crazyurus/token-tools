API说明

Token团队短网址服务提供一个API接口，可通过该API得到网址的摘要信息。

本API采用HTTP协议，返回的数据格式为JSON。

请求的地址

POST  http://token.team{:U('Tools/DWZ/add', '', '.do')}
参数 样例 说明 备注
url http://www.wutnews.net/ 需要转换的网址
title 百度 网址的说明 可选参数

返回

请求成功：

请求成功会返回状态码200，并返回摘要信息。

在摘要前拼接域名即可得到完整的短网址：http://token.team/GGqPS4。

{"code":200, "data":"GGqPS4"}
请求失败：

请求失败会返回状态码500，并返回错误信息。

{"code":500, "data":"Url的格式不正确，请检查是否含有http协议部分。"}
